Data Usage Policy for Metrou Bucuresti App

1. Types of Data Collected

1.1 Essential Data

Required for basic app functionality:

  • Device Information: OS version, app version, device model
  • App Settings: Theme preferences, language selection, accessibility settings
  • Usage Data: Features used, crash reports, error logs

1.2 Optional Data (Requires Permission)

Enhanced features that require user consent:

  • Location Data: GPS coordinates for finding nearby stations
  • Notification Preferences: For route tracking alerts

1.3 User-Generated Data

Content you create and control:

  • Personal Notes: Station notes and comments
  • Favorite Routes: Saved route preferences
  • Custom Settings: Personalized app configurations

2. How Data Is Processed

2.1 Local Processing

Most data is processed locally on your device:

  • Route calculations
  • Station searches
  • Personal notes and favorites
  • App preferences

2.2 External Services

Limited data may be shared with:

  • Location Services: For finding nearby stations (GPS data)
  • AdMob Services: For displaying relevant advertisements

3. Data Retention and Deletion

3.1 Local Data

  • Stored on your device until you delete the app
  • Can be manually cleared through app settings
  • Automatically deleted when app is uninstalled

3.2 AdMob Data

  • Advertising data managed according to Google's policies
  • You can reset your Advertising ID at any time
  • Data retention follows Google AdMob guidelines

4. Your Control Over Data

4.1 Permissions Management

You can control:

  • Location access through device settings
  • Notification permissions
  • App-level data collection preferences
  • Advertising personalization settings

4.2 Data Export and Deletion

Available through app settings:

  • Export personal notes and favorites
  • Clear all personal data
  • Reset app to factory defaults

5. Security Measures

  • Local data encryption on supported devices
  • Secure transmission protocols for any network communication
  • Regular security updates and patches
  • No storage of sensitive personal information

6. Third-Party Services

This app may integrate with:

  • Google AdMob: For displaying advertisements
  • Device Location Services: For nearby station features
  • Map Applications: When you choose to open locations in external apps

Each service has its own privacy policy and data handling practices.

7. Updates to This Policy

Policy Updates: We will notify users of any material changes to data collection or usage practices through app updates and in-app notifications. Changes will be posted on this website and within the app.

Continued use of the App after changes become effective constitutes acceptance of the updated Data Usage Policy.

8. Contact Information

Contact Us: For questions about data usage, contact us at:

  • Email: help@lxtech.ro
  • Support: Through the app's settings menu

We will respond to all data-related inquiries within a reasonable timeframe.
